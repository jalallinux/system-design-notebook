# Reference Books & Learning Resources

## Reference Books

1. **"System Design Interview - An Insider's Guide" (Volumes 1 & 2) by Alex Xu** - Primary reference for interview-style system design problems, back-of-envelope calculations, and step-by-step design frameworks.

2. **"Designing Data-Intensive Applications" by Martin Kleppmann** - Foundational reference for distributed systems theory, data storage and retrieval, replication, partitioning, transactions, consistency models, batch processing, and stream processing.

3. **"Building Microservices" by Sam Newman** - Reference for microservices architecture, service decomposition, integration patterns, and organizational aspects of distributed systems.

## Prerequisite Subjects

Students should have foundational knowledge in:
- Computer Organisation and Architecture
- Operating Systems
- Object Oriented Programming
- Software Engineering
- Computer Networks
- Database Management
- Distributed Database Management
- Big Data
